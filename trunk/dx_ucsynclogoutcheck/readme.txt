=================
概述
=================
欢迎查看Horse Luke（中文名称：微碌）所编写/修改的程序。
本程序用于检测uc同步登录是否正常（调用的是uc_user_synlogout接口）
目前在DX 1.5测试通过，理论上可用于DX 1.0/1.5/1.5.1/2.0

=================
使用方法
=================
上传复制ucsynclogoutcheck.php到论坛根目录，然后运行。
