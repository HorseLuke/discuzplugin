=================
概述
=================
欢迎查看Horse Luke（中文名称：微碌）所编写/修改的程序。
本程序用于检查现有插件目录中的插件xml文件是否是可以导入的，另外，也用于排查DX点击“安装新插件”后提示“数据无法识别”、但不提示是什么插件引起的问题。
目前可用于DX 1.5；理论上，本程序可以改写成检查风格xml是否合法。
该程序的主要使用对象：
    - 需要排查DX故障的客服
    - 部分遇到此问题的站长
    - 部分需要调试xml文件的插件开发者

=================
使用方法
=================
上传复制pluginXMLChecker.php到论坛根目录，然后运行。

如果提示“WARINING: CAN NOT PARSE THE XML FILE”：
    表示该插件的xml文件本身出现问题（比如xml文件没有写全等）。
    遇到此情况，请首先查看该xml文件内容是否正常（比如内容是否正常闭合）；或者重新下载对应插件安装包，将安装包里面的xml以FTP二进制重新传输一次。
    如果不知道怎么做，请联系插件的开发者。

如果提示“WARINING: THE XML FILE DOES NOT CONTAIN XMLNAME 'Discuz! Plugin'”：
    表示该xml文件不属于插件的。
    遇到此情况，请联系插件的开发者。
