我的附件查询（My Attachments Viewer） For Discuz! 6.x安装使用说明书
欢迎选用由Horse Luke（中文名称：竹节虚）开发的插件！

插件名称：我的附件查询（My Attachments Viewer） For Discuz! 6.x
版本：Ver 0.0.5 Build 20081119 For Discuz! 6.x(6.0.0(20081117)/6.1.0/6.1.0F)
插件有限支援链接：http://www.discuz.net/thread-1112048-1-1.html

插件说明：
该插件原模型来源于国外著名论坛程序phpBB的一个子功能：我的附件管理（User Control Panel - Manage attachments）。本插件仅抽取其核心功能——列出我所发表的附件。

 

适合的论坛版本：
Discuz! 6.0.0（需打上官方在2008-11-17发布的“Discuz! 安全补丁包下载【20081117】”（http://www.discuz.net/thread-1113736-1-1.html））
Discuz! 6.1.0
Discuz! 6.1.0F

 

安装/升级和使用方法：

请注意：
若要使得该插件在Discuz! 6.0.0下正常运行，你必须首先打上官方在2008-11-17发布的“Discuz! 安全补丁包下载【20081117】”（http://www.discuz.net/thread-1113736-1-1.html）。该补丁针对Discuz! 6.0.0不仅补漏，而且更新了数据库访问方式，使得原本只能在6.1下运行的插件可以稍加修改即可使用在6.0。

如果你是第一次使用该插件 如果你之前使用了Ver 0.0.2 Build 20081116，需要升级到Ver 0.0.5 Build 20081119 For Discuz! 6.x
上传upload文件夹的所有文件到论坛根目录。在登录状态下访问myattachments.php即可。
若想在右上角的插件上增加“ 我的附件查询”链接，请在后台导入discuz_plugin_myattachview.txt（请勾选上“允许导入不同版本 Discuz! 的插件”）。
该导入文件的作用仅增加附件查询在前台的链接显示，导入与否对此插件毫无影响。		 请首先上传upload文件夹的所有文件覆盖同名文件。
然后删除根目录的lightbox1文件夹（从0.0.5 Build 20081119开始，ajax访问图片改用Discuz!自身的显示方式，不再使用Lightbox 1）。
最后在登录状态下访问myattachments.php即可。

 

版本历史：

Ver 0.0.5 Build 20081119（当前）：
新增：正式支持Discuz! 6.0.0（需打上官方在2008-11-17发布的“Discuz! 安全补丁包下载【20081117】”（http://www.discuz.net/thread-1113736-1-1.html））
修改：对附件列表的显示方式进行了大量修改，使得显示方式更加人性化。
修正：修正存在缩略图的情况下仍读取原图的问题。
修正：查看购买记录链接含有Discuz! 7模版代码导致点击后显示javascript错误。
取消：图像显示不再使用Lightbox 1，改为Discuz!自身的显示方式。

Ver 0.0.2 Build 20081116：
新增：发布针对Discuz! 6.1.0/6.1.0F的插件版本。
特别鸣谢：图像显示部分运用了Lokesh Dhakar开发的Lightbox 1（http://www.huddletogether.com/projects/lightbox/），在此致谢！

Ver 0.0.2 Beta Build 20081115：
修正：在批量上传情况下无法正确识别文件类型，从而导致显示错误的文件类型图标，面向Discuz! 7.0.0 Beta。

Ver 0.0.1 Beta Build 20081115：
测试版本发布，面向Discuz! 7.0.0 Beta。
